ПРОБЛЕМА: Дата рождения не сохраняется при регистрации

═══════════════════════════════════════════════════════════════

ШАГ 1: ПЕРЕЗАПУСТИТЕ DEV SERVER
  
  В терминале:
  1. Ctrl+C
  2. npm run dev
  3. Дождитесь: ➜ Local: http://localhost:8080/

═══════════════════════════════════════════════════════════════

ШАГ 2: ОТКРОЙТЕ DEVTOOLS

  В браузере:
  1. Нажмите F12
  2. Перейдите на вкладку "Console"
  3. Очистите консоль (кнопка 🚫 или Ctrl+L)

═══════════════════════════════════════════════════════════════

ШАГ 3: ЗАРЕГИСТРИРУЙТЕСЬ

  1. Откройте: http://localhost:8080/auth
  2. Заполните форму регистрации
  3. ОБЯЗАТЕЛЬНО выберите дату рождения
  4. Нажмите "Зарегистрироваться"
  5. Введите код подтверждения

═══════════════════════════════════════════════════════════════

ШАГ 4: ПРОВЕРЬТЕ КОНСОЛЬ

  В консоли должно появиться:
  
  ✅ Если всё работает:
     "Updating profile with: {first_name: 'Имя', date_of_birth: '2000-01-01'}"
     "Profile update successful"
  
  ❌ Если ошибка:
     "Failed to update profile: [ошибка]"
     + Уведомление: "Регистрация успешна, но не удалось сохранить..."

═══════════════════════════════════════════════════════════════

ШАГ 5: ПОКАЖИТЕ МНЕ РЕЗУЛЬТАТ

  Сделайте скриншот консоли и покажите мне:
  - Что написано в console.log
  - Есть ли ошибки (красные строки)
  - Какой запрос отправился (вкладка Network)

═══════════════════════════════════════════════════════════════

ДОПОЛНИТЕЛЬНО: Проверьте Network

  1. В DevTools перейдите на вкладку "Network"
  2. Зарегистрируйтесь
  3. Найдите запрос PUT /api/profile
  4. Кликните на него
  5. Посмотрите:
     - Request Payload (что отправили)
     - Response (что вернул backend)
  6. Покажите мне скриншот
